# 👶 婴儿哭声识别 (Baby Cry Recognition)

🦞 一个能识别婴儿哭声类型的智能应用

## 项目状态

🚧 开发中 - Phase 1: 架构设计

## 功能特性

- 🎤 实时音频监控
- 🧠 AI 哭声分类（饥饿/困倦/不适/正常）
- 📱 iOS & Android 双平台支持
- 🔔 智能通知推送
- 📊 历史记录和统计
- ⚙️ 可选后台持续监听
